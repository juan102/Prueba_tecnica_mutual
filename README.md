## Cómo compilar

Para compilar y ejecutar este proyecto, necesitarás:

*   **Java 17**: Asegúrate de tener instalado el JDK 17 o una versión superior.
*   **Gradle**: El proyecto utiliza el Gradle Wrapper, por lo que no necesitas instalar Gradle por separado.

Para compilar el proyecto, ejecuta el siguiente comando en la raíz del proyecto:

```bash
./gradlew build
```

Para ejecutar el proyecto, puedes usar:

```bash
./gradlew bootRun
```
